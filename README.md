# Vanilla WordPress composer-based boilerplate

This boilerplate used for build template of Wodby 2.0's [WordPress PHP service](https://github.com/wodby/service-wordpress-php)

A Wodby CI build pipeline included for Wodby 2.0

See http://wodby.com/stacks/wordpress

- This repository uses composer to manage WordPress dependencies
- We use https://wp-packages.org to install WordPress core
- WordPress will be installed in the `web` directory.
